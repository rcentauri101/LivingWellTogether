# Meal Plan Site

An Astro-powered static site for sharing a three-month meal plan, weekly menus, and health tips with friends.

## Getting started

1. Install dependencies with `npm install`
2. Start the dev server with `npm run dev`
3. Build the static site with `npm run build`

## Project structure

- `src/data/meals.json`: master meal list
- `src/data/weekly-plans.json`: weekly breakfast, lunch, and dinner selections
- `src/data/health-tips.json`: shared wellness tips
- `src/pages/`: site pages
- `src/components/`: reusable Astro components

## GitHub Pages

The Astro config is set up for a GitHub Pages project site hosted at:

`https://livingwelltogether.github.io`

If your GitHub username or repository name changes, update `site` and `base` in `astro.config.mjs`.
